---
name: diff
description: Show the current git diff, status, and recent changes
---

Show the current state of the repository:

1. Run `git status` to show the working tree status
2. Run `git diff` to show unstaged changes
3. Run `git diff --staged` to show staged changes
4. If there are recent commits on the current branch, show them with `git log --oneline -10`
5. Present all results clearly, highlighting what files changed and the nature of changes
